Windows Desktop · Java 21 + JavaFX

Duas formas de entrega: dist local (JAR + run.bat + lib — exige JDK 21 no PATH) ou instalador .exe (autocontido, sem Java externo).

Dist local (oficina)

Na oficina (caracore-ink): cd agenda-java && mvn clean package -DskipTests. Gera target/dist/ com agenda.jar, lib/ (JavaFX, SQLite) e run.bat. Executar: target\dist\run.bat. Requer JDK 21 no PATH. O run.bat inclui os módulos javafx.controls, javafx.fxml e javafx.web (Splash com WebView).

Alternativa sem dist: mvn javafx:run na pasta agenda-java.

Instalador Windows (.exe)

Na oficina: mvn clean package -Pinstaller. Gera o instalador em target/installer/. Executável autocontido (jpackage inclui runtime); não exige Java instalado. Janela sem console (winConsole=false); ícone oficial em src/main/resources/icons/agenda-ink.ico.
