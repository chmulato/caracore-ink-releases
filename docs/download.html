Verificação de integridade (Windows ZIP + macOS DMG + Linux DEB)

No PowerShell ou terminal, na pasta onde os arquivos foram salvos:

# SHA256 (recomendado) — Windows RC3
(Get-FileHash .\AgendaInk-2.0.0-RC3-windows.zip -Algorithm SHA256).Hash
# Valor esperado (Windows):
# c25b6ee8e5853a2a31aafba387aede31fb51bd0b6a9368b7343c7cb9ea2039a9

# SHA256 (recomendado) — macOS RC3
shasum -a 256 ./AgendaInk-2.0.0-RC3.dmg
# Valor esperado (macOS):
# 57f5bc7257204b352389f51db231707408706f5d01102a6cafc86b2de112ca45

# SHA256 (recomendado) — Linux RC3

# Valor esperado (Linux):
# 8574c57112d652e1e944b1bdca6145b772dfb63111a8075ede4897e558aa44d7

Todos os artefatos são gerados com bytecode ofuscado via ProGuard — proteção do código-fonte na distribuição.

Como usar o ZIP Windows (modo portable)

  1. Baixe e salve AgendaInk-2.0.0-RC3-windows.zip.
  2. Descompacte (clique direito → Extrair aqui).
  3. Abra a pasta AgendaInk\.
  4. Execute AgendaInk.exe com duplo clique.
  5. Na primeira execução o banco de dados será criado em:
    %LOCALAPPDATA%\CaraCore\AgendaInk\agenda.db
