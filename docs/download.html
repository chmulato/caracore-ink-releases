Download — Cara Core Ink Agenda

Windows Desktop · Java 21 + JavaFX

Requisito: JDK 21 instalado no Windows (OpenJDK ou Oracle). O aplicativo é distribuído como JAR.

O Cara Core Ink Agenda é distribuído como JAR para execução no Windows. É necessário ter o JDK 21 (Java 21) no PATH. A aplicação usa JavaFX 21 para a interface gráfica.

Como executar (desenvolvimento):

  • Na oficina (caracore-ink): cd agenda-java && mvn javafx:run
  • Ou gerar JAR: mvn package — o JAR fica em agenda-java/target/
