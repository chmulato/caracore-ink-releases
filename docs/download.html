Verificação de integridade (RC8)

No PowerShell ou terminal, na pasta onde os arquivos foram salvos:

# Windows
(Get-FileHash .\AgendaInk-2.0.0-RC8-windows.exe -Algorithm SHA256).Hash
(Get-FileHash .\AgendaInk-2.0.0-RC8-windows.zip -Algorithm SHA256).Hash

# Linux/macOS
sha256sum AgendaInk-2.0.0-RC8-linux.deb
shasum -a 256 AgendaInk-2.0.0-RC8-macos.dmg

# Compare com o arquivo checksum.sha256 publicado na release RC8.

Todos os artefatos são gerados com bytecode ofuscado via ProGuard — proteção do código-fonte na distribuição.

Instalação padrão no Windows 10/11

  1. Baixe AgendaInk-2.0.0-RC8-windows.exe.
  2. Opcionalmente, valide o hash com o checksum.sha256 da release.
  3. Execute o instalador e avance no assistente.
  4. Abra o app pelo Menu Iniciar (atalho AgendaInk).

Como usar o ZIP Windows (modo portable)

  1. Baixe e salve AgendaInk-2.0.0-RC8-windows.zip.
  2. Descompacte (clique direito → Extrair aqui).
  3. Abra a pasta AgendaInk\.
  4. Execute AgendaInk.exe com duplo clique.
  5. Na primeira execução o banco de dados será criado em:
    %LOCALAPPDATA%\CaraCore\AgendaInk\agenda.db
