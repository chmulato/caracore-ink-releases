Verificação de integridade (RC7)

No PowerShell ou terminal, na pasta onde os arquivos foram salvos:

# SHA256 (recomendado) — Windows RC7
(Get-FileHash .\AgendaInk-2.0.0-RC7-windows.zip -Algorithm SHA256).Hash

# SHA256 oficial do Windows RC7
eda86861c480f00da097d49c1d90dbcacfc65880b968d84dbfae0b6f6faac253

# Consulte os arquivos checksum.sha256 e checksum.md5 na release RC7.
# Linux, macOS e iPhone: valide pelos checksums quando os artefatos forem publicados.

Todos os artefatos são gerados com bytecode ofuscado via ProGuard — proteção do código-fonte na distribuição.

Como usar o ZIP Windows (modo portable)

  1. Baixe e salve AgendaInk-2.0.0-RC7-windows.zip.
  2. Descompacte (clique direito → Extrair aqui).
  3. Abra a pasta AgendaInk\.
  4. Execute AgendaInk.exe com duplo clique.
  5. Na primeira execução o banco de dados será criado em:
    %LOCALAPPDATA%\CaraCore\AgendaInk\agenda.db
