Conteúdo alinhado entre matriz e vitrine

  • index.html — Apresentação do Ink Agenda (agenda, finanças, painel de resultados para estúdios de tatuagem), aviso padrão de lançamento (26 de junho de 2026), links para Download, Tecnologia, Canal de feedback, Portfólio e Vitrine.
  • download.html — Requisitos, artefatos RC1 Windows e Linux, links para GitHub Releases. Guias de instalação por plataforma. Aviso padrão de lançamento em todas as páginas.
  • tecnologia.html — Stack: Java 21, JavaFX 21, Maven, SQLite WAL, ProGuard, jpackage; arquitetura em camadas; build e testes (203 testes, JaCoCo).
  • canal-feedback.html — E-mail suporte@caracore.com.br, WhatsApp e Telegram (assunto Cara Core Ink Agenda).
  • manual.html — Guia de uso: login, agenda, financeiro, painel, backup.

Em ambos (matriz e vitrine), o aviso padrão é o mesmo: App Cara Core Ink Agenda para Windows Desktop — 26 de junho de 2026. A mensagem e a data estão alinhadas. A matriz é a fonte de apresentação completa e consultoria; a loja é a vitrine pública e o ponto de download.

Links cruzados

Estilo e CSS

O portal usa assets/css/ink-portal.css, alinhado ao padrão das lojas do ecossistema (Hub, RU, PDV):

  • Tema escuro · Fonte: Inter (Google Fonts)
  • Variáveis: --accent-blue, --accent-gold, --accent-green
  • Componente .ink-launch-banner com borda verde e data de lançamento
  • Componente .manual-card para seções de conteúdo (usado neste portal)
  • Componente .cta-btn / .cta-btn-primary / .cta-btn-secondary para ações
