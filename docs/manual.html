Visão geral

O Cara Core Ink Agenda é um aplicativo desktop para Windows que ajuda tatuadores e estúdios a organizar agendamentos, controlar entradas e saídas e acompanhar o desempenho do negócio. Os dados ficam no seu computador; a interface é nativa (JavaFX), sem depender de navegador.

Principais recursos

  • Agenda de sessões — Cliente, data, horário e valor. Visualização em calendário e lista.
  • Controle financeiro — Registro de receitas (entradas) e despesas (saídas), com saldo por período.
  • Painel de resultados — Faturamento, médias e indicadores para acompanhar o desempenho.

O aplicativo é desenvolvido em Java 21 e JavaFX 21. Requisitos e detalhes técnicos estão em Download e Tecnologia.

Requisitos e instalação

Requisito: ter o JDK 21 (Java 21) instalado no Windows, com java no PATH. O aplicativo é distribuído como JAR.

Como obter o aplicativo: acesse a página de Download e use o link para Releases → Latest (GitHub). Baixe o JAR e execute com java -jar nome-do-arquivo.jar (ou duplo clique, se o Windows estiver configurado para abrir JARs com Java).

Para desenvolvimento na oficina (código-fonte): cd agenda-java && mvn javafx:run. Ver Download para mais detalhes.

Agenda de sessões

A agenda permite cadastrar e visualizar agendamentos de sessões: nome do cliente, data, horário (opcional) e valor (opcional). Use para planejar o dia e a semana e manter o histórico de atendimentos.

O que você pode fazer

  • Criar agendamento — Informe cliente, data e, se quiser, horário e valor. Salve para que a sessão apareça na agenda.
  • Editar agendamento — Abra o agendamento e altere os dados (cliente, data, horário, valor) conforme necessário.
  • Listar e filtrar — Visualize os agendamentos por período (dia, semana, mês) para organizar a rotina.

O aplicativo mantém os dados localmente. Use backups do sistema de arquivos ou da pasta de dados do app se precisar preservar o histórico.

Controle financeiro

O módulo financeiro registra entradas (receitas) e saídas (despesas), calculando o saldo. Ideal para controlar o que entra (sessões, retoques, vendas) e o que sai (materiais, aluguel, contas).

Tipos de lançamento

  • Entrada (receita) — Valores que entram: sessão de tatuagem, retoque, venda de produtos, etc.
  • Saída (despesa) — Valores que saem: materiais, aluguel, energia, manutenção, etc.

Uso básico

  • Registre cada receita ou despesa com descrição, valor e data.
  • O aplicativo calcula o saldo (receitas menos despesas) no período.
  • Consulte a lista de lançamentos para revisar ou editar.

Painel de resultados

O painel reúne indicadores do negócio: faturamento no período, totais de entradas e saídas, saldo e médias (por exemplo, ticket médio). Use para acompanhar o desempenho mensal e tomar decisões.

Conforme novas versões forem lançadas, o painel pode incluir gráficos e relatórios por período. A apresentação completa e a consultoria estão na matriz (caracore.com.br).

Problemas comuns

  • O JAR não abre — Verifique se o JDK 21 está instalado e no PATH. No terminal: java -version deve mostrar versão 21. Reinstale o JDK 21 (OpenJDK ou Oracle) se necessário.
  • Erro ao iniciar a janela — Confirme que está usando Java 21 e que o JavaFX está disponível (o JAR de release já inclui o necessário). Em caso de erro de memória, tente aumentar a heap: java -Xmx512m -jar nome-do-arquivo.jar.
