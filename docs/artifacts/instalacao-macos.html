Instalação passo a passo

  1. Baixe o arquivo AgendaInk-2.0.0-RC1.dmg.
  2. Verifique o checksum SHA256 publicado na página de release.
  3. Abra o arquivo .dmg com duplo clique.
  4. Na janela que abrir, arraste AgendaInk.app para a pasta Aplicativos.
  5. Ejete o volume DMG.
  6. Abra o Finder → Aplicativos e execute AgendaInk.

Nota de segurança macOS: Na primeira execução, o macOS pode exibir alerta de Gatekeeper por ser software de desenvolvedor não notarizado. Acesse Ajustes do Sistema → Privacidade e Segurança e clique em "Abrir mesmo assim". O app não requer conexão com a internet.
