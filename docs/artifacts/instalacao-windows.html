Verificação de integridade

Antes de instalar, confirme que o arquivo não foi corrompido. No PowerShell, na pasta onde o ZIP foi salvo:

SHA256 (recomendado)

(Get-FileHash .\AgendaInk-2.0.0-RC3-windows.zip -Algorithm SHA256).Hash

Valor esperado:

C25B6EE8E5853A2A31AAFBA387AEDE31FB51BD0B6A9368B7343C7CB9EA2039A9

Via CertUtil (Prompt de Comando)

certutil -hashfile AgendaInk-2.0.0-RC3-windows.zip SHA256

Instalação passo a passo

  1. Baixe o arquivo AgendaInk-2.0.0-RC3-windows.zip para uma pasta de sua escolha.
  2. Verifique os checksums conforme seção acima.
  3. Clique com o botão direito no ZIP → Extrair aqui (ou use 7-Zip, WinRAR).
  4. Abra a pasta AgendaInk\ extraída.
  5. Execute AgendaInk.exe com duplo clique.
  6. Na primeira execução, o banco de dados é criado automaticamente em:
    %LOCALAPPDATA%\CaraCore\AgendaInk\agenda.db

Nota RC3: Esta versão é o Release Candidate 3 para homologação, entregue como ZIP portable com runtime Java 21 embutido.

Aviso de antivírus / SmartScreen

Por ser um artefato pré-release sem assinatura de código comercial, o Windows SmartScreen ou seu antivírus pode exibir um aviso na primeira execução. Isso é esperado para software não assinado.

  • Na janela do SmartScreen: clique em "Mais informações" → "Executar mesmo assim".
  • Verifique sempre os checksums antes de executar qualquer arquivo baixado.

Localização dos dados

Desinstalação

Esta versão RC3 é portable (APP_IMAGE). Para remover:

  1. Exclua a pasta AgendaInk\ extraída do ZIP.
  2. Opcionalmente, remova os dados em %LOCALAPPDATA%\CaraCore\AgendaInk\.

O instalador com wizard e desinstalador registrado no Windows chegará no lançamento oficial (26/06/2026).
