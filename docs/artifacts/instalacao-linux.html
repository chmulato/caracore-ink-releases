Instalação passo a passo (prévia)

Após baixar o arquivo .deb:

Opção 1 — dpkg (recomendado)

sudo dpkg -i AgendaInk-2.0.0-linux.deb
sudo apt-get install -f   # resolve dependências, se necessário

Opção 2 — apt

sudo apt install ./AgendaInk-2.0.0-linux.deb

Após instalar, execute o aplicativo pelo menu ou pelo terminal:

agendaink

Runtime embutido: O pacote inclui o runtime Java 21 — não é necessário instalar JDK/JRE separadamente. O app roda de forma isolada.

Verificação de integridade

Os checksums SHA256 serão publicados junto com o artefato no GitHub Releases. Para verificar no terminal:

# SHA256
sha256sum AgendaInk-2.0.0-linux.deb

# MD5
md5sum AgendaInk-2.0.0-linux.deb

Compare o hash calculado com o publicado na página de release antes de instalar.

Localização dos dados (prévia)

Item Caminho
Banco de dados ~/.local/share/CaraCore/AgendaInk/agenda.db
Backups automáticos ~/.local/share/CaraCore/AgendaInk/
Executável /opt/agendaink/bin/agendaink

Desinstalação

sudo apt remove agendaink
# ou
sudo dpkg -r agendaink

Os dados do banco permanecem em ~/.local/share/CaraCore/AgendaInk/ e podem ser removidos separadamente.
