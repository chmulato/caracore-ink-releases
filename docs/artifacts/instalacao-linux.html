Instalação — Linux

Cara Core Ink Agenda v2.0.0-RC1 · Ubuntu 20.04 LTS+ / Debian 11+ · 64 bits · Artefato disponível

Requisitos do sistema

  • Ubuntu 20.04 LTS ou superior (Focal, Jammy, Noble) — ou distribuição Debian-compatível
  • Arquitetura x86-64 (amd64)
  • Aproximadamente 400 MB de espaço livre em disco
  • Nenhuma instalação de Java necessária — runtime Java 21 embutido no pacote
  • Suporte a bibliotecas GTK3 (padrão no Ubuntu/Debian — normalmente já instalado)
  • Conexão com internet: não necessária para uso do aplicativo

Instalação passo a passo

Após baixar o arquivo .deb:

Opção 1 — dpkg (recomendado)

sudo dpkg -i agenda-ink_2.0.0_amd64.deb
sudo apt-get install -f   # resolve dependências, se necessário

Opção 2 — apt

sudo apt install ./agenda-ink_2.0.0_amd64.deb

Após instalar, execute o aplicativo pelo menu ou pelo terminal:

agendaink

Runtime embutido: O pacote inclui o runtime Java 21 — não é necessário instalar JDK/JRE separadamente. O app roda de forma isolada.

Desinstalação

sudo apt remove agendaink
# ou
sudo dpkg -r agendaink

Os dados do banco permanecem em ~/.local/share/CaraCore/AgendaInk/ e podem ser removidos separadamente.
