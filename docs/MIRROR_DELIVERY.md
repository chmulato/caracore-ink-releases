# Espelho de delivery — Cara Core Ink Agenda

Este diretório é o **portal de delivery do Cara Core Ink Agenda** no domínio da **Cara Core Informática (matriz)**.

## Onde está o quê

| Local | Papel | URL (exemplo) |
|-------|--------|----------------|
| **Matriz (este diretório)** | Portal completo de delivery no site Cara Core | `https://caracore.com.br/delivery/ink/` |
| **Vitrine (loja)** | Repositório público de vitrine e releases | `https://ink.caracore.com.br/` |

## Conteúdo alinhado

- **index.html** — Apresentação do Ink Agenda (agenda, finanças, painel de resultados para estúdios de tatuagem), aviso padrão de lançamento (26 de junho de 2026), links para Download, Tecnologia, Canal de feedback, Portfólio e Vitrine (caracore-ink-releases).
- **download.html** — Requisitos (JDK 21), JAR, link para GitHub Releases (releases/latest). Aviso padrão de lançamento em todas as páginas.
- **tecnologia.html** — Stack: Java 21, JavaFX 21, Maven; arquitetura em camadas; build e testes (JaCoCo).
- **canal-feedback.html** — E-mail suporte@caracore.com.br, WhatsApp e Telegram (assunto Cara Core Ink Agenda).

Em ambas (matriz e vitrine), o **aviso padrão** é o mesmo: **App Cara Core Ink Agenda para Windows Desktop — 26 de junho de 2026.** A mensagem e a data estão alinhadas; a matriz é a fonte de apresentação completa e consultoria; a loja é a vitrine pública e o ponto de download. A matriz foi atualizada com as novidades da loja: pilares (Agenda, Controle financeiro, Painel de resultados), bloco CTA (“Fale com a Cara Core”), download com requisito JDK 21 e comandos de desenvolvimento, tecnologia com selo Ecossistema e roadmap (Próximos passos).

## Links cruzados

- Na **matriz**: breadcrumb e footer apontam para o **ecossistema (Garagem)** (caracore.com.br/ecosistema.html#garagem). Ink entra no portfólio após o lançamento. Vitrine: ink.caracore.com.br. Links internos para index, download, tecnologia e canal-feedback.
- Na **vitrine**: footer e nav apontam para o **portfólio** e para a matriz (caracore.com.br/delivery/ink/) quando aplicável; link para Releases no GitHub.

## Estilo

O portal usa `assets/css/ink-portal.css`, alinhado ao padrão das lojas do ecossistema (Hub, RU, PDV): tema escuro, Inter, variáveis --accent-blue, --accent-gold, --accent-green. Banner de lançamento (ink-launch-banner) com borda verde e data **26 de junho de 2026**.

## Lançamento programado

- **Data de lançamento oficial:** 26 de junho de 2026.
- **Produto:** App Cara Core Ink Agenda para Windows Desktop (Java 21, JavaFX 21).
- Oficina: caracore-ink · Loja: caracore-ink-releases.
